## 按阶段查找

### 阶段 2：基础统计分析
- **`05`** → 如何正确解读频率分布；关注分布形态而非只看 Top3；避免均值误导

### 阶段 3：交叉分析
- **`05`** → 差异显著性判断标准（结合样本量）；李克特量表的正确解读；NPS 差异的判断阈值

### 阶段 4：文本分析（AI 分析步骤）
- **`01`** → 主题六步归纳流程；维度质量检查标准（充分性/独立性/相关性）
- **`02`** → 聚类操作规范；不预设分类；大维度拆分；保留异常点
- **`04`** → 强度信号识别；行为vs表述矛盾识别；开放题专用技巧

### 阶段 5：综合报告撰写
- **`03`** → 综合证据的标准写法；多源分歧时的处理方式
- **`06`** → 定性定量融合的三种整合策略；综合洞察写作结构模板
- **`07`** → 发现明显用户分群时的画像描述方法（按需）
- **`08`** → 策略建议的量化支撑；机会优先级排序；范围值写法规范
